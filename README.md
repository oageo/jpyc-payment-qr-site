# jpyc-payment-qr-site
![GitHub License](https://img.shields.io/github/license/oageo/jpyc-payment-qr-site)

特定額のJPYCの支払いを求めるQRコードをクライアントサイドで生成するツール

このツールは「[jpyc-payment-qr](https://github.com/oageo/jpyc-payment-qr)」を基に動作しています。

* 生成されたQRコード・URIの内容は送金前に必ずご自身で確認してください。誤った宛先・金額の送金について、作者は一切の責任を負いません。
* ブロックチェーン上のトランザクションは原則として取り消しができません。
* テストネット機能はデバッグ目的で提供されます。本番環境での利用は本番ネットワークを選択してください。
